# neulbo-ui


# 🦥 늘보(Neulbo) UI 퍼블리싱 

'늘보' 프로젝트의 HTML/CSS 마크업 및 디자인 시안을 관리하는 곳
**어스 톤(Earth Tone)** 팔레트를 바탕으로 여유롭고 편안한 UI

---

## 🎨 컬러 팔레트 (Earth Tone Theme)

프로젝트 전반에 사용될 핵심 색상 리스트입니다.

| 구분 | 색상 코드 | RGB | 프리뷰 | 용도 |
|:---:|:---:|:---:|:---:|:---|
| **Background** | `#FAF9EE` | `250, 249, 238` | 🟡 | 메인 배경색 (따뜻한 아이보리) |
| **Point (Primary)** | `#A2AF9B` | `162, 175, 155` | 🟢 | 포인트 컬러, 메인 버튼, 체크박스 (세이지 그린) |
| **Point (Sub)** | `#DCCFC0` | `220, 207, 192` | 🟤 | 보조 포인트, 카드 테두리, 강조 (샌드 베이지) |
| **Surface** | `#EEEEEE` | `238, 238, 238` | ⚪ | 입력창 배경, 보조 UI 요소 (라이트 그레이) |
